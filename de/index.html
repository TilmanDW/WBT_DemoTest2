🤖 KI-Modellinformationen

Modell: Mistral-7B-Instruct-v0.3

Entwickler: Mistral AI

Anbieter: Hugging Face Inference API

Parameter: 7,3 Milliarden Parameter, optimiert für Instruktionsbefolgung

Version: v0.3 (September 2024) - Neueste stabile Version

Fähigkeiten: Erweiterte Textzusammenfassung, Metadaten-Extraktion, kreatives Umschreiben, mehrsprachige Unterstützung (Deutsch, Englisch, Französisch, Spanisch, Italienisch)

Einschränkungen: Maximale Eingabelänge ~8000 Token (~6000 Wörter). Antwortqualität hängt von der Eingabeklarheit ab. Kostenlose API-Stufe beinhaltet Ratenbegrenzung zu Stoßzeiten. Verarbeitungszeit: 2-10 Sekunden je nach Textlänge und Serverlast.

Fallback: Demo-Modus aktiviert sich automatisch, falls die API nicht verfügbar ist, um konsistente Funktionalität zu gewährleisten.
