Version: v0.3 (September 2024) - Neueste stabile Version

Fähigkeiten: Erweiterte Textzusammenfassung, Metadaten-Extraktion, kreatives Umschreiben, mehrsprachige Unterstützung (Deutsch, Englisch, Französisch, Spanisch, Italienisch)

Fallback: Demo-Modus aktiviert sich automatisch, falls die API nicht verfügbar ist, um konsistente Funktionalität zu gewährleisten.
